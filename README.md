# Accident Prevention Safety System using Arduino

## Overview

This project was developed at **Indian Institute of Science Education and Research (IISER) Pune** under the guidance of **Dr. Shouvik Datta**.

The objective of this project was to design an intelligent accident prevention system capable of detecting approaching vehicles at blind road intersections using infrared sensing technology and providing timely warning signals to reduce collision risks.

---

## Features

- Vehicle detection using IR sensors
- Speed estimation of approaching vehicles
- Warning system for overspeeding vehicles
- Real-time embedded implementation using Arduino
- Low-cost traffic safety solution

---

## Hardware Used

- Arduino Uno

- LEDs / Display Module
